### Config
To run on local machine change the backed url in `/client/src/backendip.js`
```javascript
export default 'your backend url'
```

## Available Scripts

In the project directory, you can run:

### `npm run frontend`

Runs React in development mode

### `npm run backend`

Runs Express server.
